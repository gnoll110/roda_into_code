# roda_intro_code
Roda framework intro code sample

To run the code on the front page of the [project's website][1].

## Install & run

```
> git clone https://github.com/gnoll110/roda_intro_code.git
> cd roda_intro_code
> bundle install
..................
> bundle exec rspec
> bundle exec rackup
```
